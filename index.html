Crie um programa que leia 5 nomes de pessoas, em seguida o programa deverá imprimir esses 5 nomes em Letras maiúsculas. Sabemos que as linguagens de programação oferecem alguns recursos chamados bibliotecas que permitem expandir a capacidade das linguagens. O portugol possuí uma biblioteca chamada Texto que permite o uso de uma função chamada caixa_alta() para converter textos para letras maiúsculas.

Agora você deverá criar um programa unindoas atividades 1, 2, 5, 6. Cada uma das atividades desenvolvidasdeveráser uma função, o programa deverá ter um menu para que o usuário possa escolher qual função/atividadeele deseja executar. De acordo com a opção escolhida o programa deverá executar completamente todo o algoritmo sem ser necessário passar parâmetros e retornar valores. Ao final da execução de cada programa você deverá perguntar ao usuário se ele quer executar novamente aquela função ou se ele deseja voltar ao menu para executar outra opção.Além disto no menu você deve criar uma opção para ele sair do programa.
